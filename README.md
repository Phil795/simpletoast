# simpletoast
Einfaches vanilla JS Bootstrap5 Toast plugin.

## Usage
Die Types basieren auf den Bootstrap Farben:
- light (default)
- dark
- info
- success
- warning
- danger

```js
SimpleToast.show('message', 'title=""', 'type=light', 'durationInSeconds=5');
```
